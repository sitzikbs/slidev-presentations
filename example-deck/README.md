# Example Deck

This is an example deck demonstrating all available layouts and features of the dark-geometric Slidev theme.

## Features Demonstrated

- Cover slide with geometric background
- Section dividers (Updates layout)
- Standard content slides
- Figure layout for scientific visualizations
- Grid layouts (grid-cols, media-grid) for comparing multiple plots
- Code highlighting with VS Code Dark Plus theme
- Google Drive media embedding
- Progress bar navigation
- Automatic page numbering

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Theme

This deck uses the custom dark-geometric theme located in `../theme/`.
