# Slidev Deck Template

This is a template for creating new Slidev presentation decks.

## Do not use this template directly!

Use the `create-deck.sh` script from the root directory to create a new deck from this template.

## Structure

- `slides.md` - Main presentation file
- `components/` - Custom Vue components
- `pages/` - Additional slide pages to import
- `snippets/` - Code snippets for demos

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Theme

This template uses the custom dark geometric theme located in `../theme/`.
